# 🏋️ Fitness Tariffs — Тестовое задание

Интерфейс выбора тарифов для фитнес-приложения с таймером, анимациями и адаптивной вёрсткой.

[🔗 Демо](https://tariffs-test.vercel.app/)

---

## 🚀 Технологии

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Fonts**: Google Fonts (Montserrat, Raleway, Manrope)
- **Package Manager**: npm / pnpm / yarn

---

## 📦 Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/vvkon13/tariffs-test.git
cd tariffs-test

# 2. Установите зависимости
npm install
# или
pnpm install
# или
yarn install
